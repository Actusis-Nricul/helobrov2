## 🚀 Usage

### Basic Setup
1. Copy the desired script
2. Inject using your Roblox executor
3. Paste the script into the executor console
4. Execute and follow in-game prompts

### Example: Running Piggy Autofarm
```lua
-- Load the script into your executor
loadstring(readfile("PiggyAutofarm"))()
-- Follow instructions to activate
```
